## Why graph-grammar

<CardGrid stagger>
  <Card title="Framework-agnostic" icon="puzzle">
    The engine has no DOM or framework dependencies , just `zod`. Use it in Node,
    the browser, or a worker.
  </Card>
  <Card title="Fast" icon="rocket">
    O(1) label buckets, an allocation-free VF2-style matcher, and single-match
    rewriting keep parallel-growth grammars **O(N)** , ~116,000 steps/s to 50k nodes.
  </Card>
  <Card title="Expressive rules" icon="setting">
    Property predicates, negative application conditions, edge embedding, property
    expressions, and four application strategies.
  </Card>
  <Card title="Validated I/O" icon="approve-check">
    Import and export grammars as JSON with runtime `zod` validation built in.
  </Card>
</CardGrid>

## Install

```sh
npm install graph-grammar
```
